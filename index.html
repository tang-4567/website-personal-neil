Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur halaman web. HTML memungkinkan kita untuk menambahkan teks, gambar, tautan, dan elemen lainnya ke dalam halaman web.

Contoh Elemen HTML

  • Heading: Gunakan tag <h1> hingga <h6> untuk judul.
  • Paragraf: Gunakan tag <p> untuk menulis teks paragraf.
  • Tautan: Gunakan tag <a> untuk membuat hyperlink.
  • Gambar: Gunakan tag <img> untuk menambahkan gambar.

Contoh Kode

        <!DOCTYPE html>
        <html>
        <head>
            <title>Contoh HTML</title>
        </head>
        <body>
            <h1>Halo Dunia!</h1>
            <p>Ini adalah contoh paragraf.</p>
            <a href="https://www.example.com">Kunjungi Situs</a>
        </body>
        </html>
